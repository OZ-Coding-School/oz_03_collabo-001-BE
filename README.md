# dev mode for docker  

### 1. 도커 데스크탑 설치  
[도커사이트](https://www.docker.com/)

### 2. 깃클론  
```bash
git clone https://github.com/OZ-Coding-School/oz_03_collabo-001-BE.git
```

### 3. 프로젝트 폴더로 이동  
```bash
cd oz_03_collabo-001-BE
```

### 4. 독커 컴포즈 실행
```bash
docker-compose -f docker-compose-dev.yml up
```
